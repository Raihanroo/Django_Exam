1. Project Overview

🎯 Project Purpose

এটি একটি Product Inventory Management System যেখানে:

  • Bulk Excel upload দিয়ে product import করা যায়
  • Draft থেকে Approved workflow আছে
  • Search এবং Pagination সাপোর্ট করে
  • Real-time status tracking সম্ভব

Technology Stack:

Django 5.2 Python 3.11 SQLite/PostgreSQL Pandas openpyxl HTML/CSS

2. Database Design

💡 Real World Context

একটি inventory system এ product data সংরক্ষণ করতে হয়। প্রতিটি product এর unique ID, name, category, price, quantity থাকে। Status field দিয়ে বুঝায় product এখনো draft অবস্থায় আছে নাকি approved হয়ে inventory তে যোগ হয়েছে।
